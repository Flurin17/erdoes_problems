# Literature and Source Notes for Erdős Problem #881

## Source page: erdosproblems.com/881

Accessed locally on 2026-05-31.

Problem statement:

> Let \(A\subset\mathbb N\) be an additive basis of order \(k\) which is
> minimal, in the sense that if \(B\subset A\) is any infinite set then
> \(A\setminus B\) is not a basis of order \(k\). Must there exist an
> infinite \(B\subset A\) such that \(A\setminus B\) is a basis of order
> \(k+1\)?

The page currently marks the problem as **open**. It lists tags "number
theory" and "additive basis".

The page has a formalized Lean statement in the Google DeepMind
`formal-conjectures` repository. That statement interprets "additive basis"
as an **asymptotic additive basis**, i.e. all sufficiently large natural
numbers are represented.

3. natso26 replies that a standard check found several major gaps in the

The problem page's status widget says there are no solutions, partial or
complete, incorporated into the remarks. Therefore the comment is useful only
as a source of possible ideas, not as evidence.

The linked Overleaf project was downloaded locally on 2026-05-31 and checked
against the \(k=2\) recurrence lemmas in `PROOF.md`. Its proposed negative
construction would give an order-2 basis with infinitely many singleton
order-3 private witnesses. Lemma 8, Corollary 8.1, and Theorem 8.2 in this
workspace show that such a configuration instead forces an infinite deletion
that remains an order-3 basis. Thus the linked construction cannot be used as
written. The apparent local error is that its affine-avoidance witness lemma
does not also prove compatibility with the already-built lower coverage:
for a private witness \(p\), every retained padder \(e\) with \(p-e\) in the
covered range must have all two-term representations of \(p-e\) pass through
the deleted element. Avoiding finitely many formal affine equations in a
fresh box does not by itself establish this shifted domination once the old
set covers a tail.

## Working literature facts to verify later

These are standard-looking background items that should be verified from
primary sources before being used in a final proof.

* Essential elements of asymptotic bases: if \(A\) is an asymptotic basis,
  an element \(a\in A\) is essential if \(A\setminus\{a\}\) is not an
  asymptotic basis of any finite order. Classical results imply only
  finitely many essential elements.
* Essential subsets and the primitive-set transform of Deschamps and Farhi
  may be relevant to finite deletions.
* Minimal asymptotic bases of order \(h\) exist for all \(h\ge 2\), and many
  constructions create private witnesses for each element at order \(h\).
  What is needed here is stronger: private witnesses that survive against
  order \(h+1\) after deletion, or a theorem preventing such witnesses.
