# Computation for Erdős 1005

- `search_a.py`: Farey-successor recurrence plus a sliding window and range-maximum structure, (O(|F_n|\log n)). Its built-in oracle independently enumerates and sorts fractions for small orders.
- `search_b.py`: independently enumerates and cross-multiplication-sorts reduced fractions, then uses a two-dimensional Fenwick maximum; it includes a direct quadratic oracle.
- `search_a_4_500.tsv`: all exact maximizers for every (4\le n\le500), generated by implementation A.

Reproduction:

```text
python3 computational/search_a.py --self-test 40
python3 computational/search_a.py --range 4 500 --output computational/search_a_4_500.tsv
python3 computational/search_b.py --min-n 4 --max-n 200 --verify-up-to 25 --only-records
python3 computational/search_a.py --orders 50 100 200 500 1000 2000 5000 --show-maximizers
python3 computational/verify_construction.py --max-n 500
```

The first command checks the recurrence generator against exhaustive enumeration and the fast extremum against a direct all-pairs checker for every order through 40. Numerical agreement and lack of counterexamples are not proofs of the asymptotic claim.
