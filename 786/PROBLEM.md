# Erdős Problem #786

Source: <https://www.erdosproblems.com/786>  
Fetched from the site's LaTeX-source view on 2026-07-16.

## Verbatim statement

Let $\epsilon>0$. Is there some set $A\subset\mathbb N$ of density $>1-\epsilon$ such that $a_1\cdots a_r=b_1\cdots b_s$ with $a_i,b_j\in A$ can only hold when $r=s$?

Similarly, can one always find a set $A\subset\{1,\ldots,N\}$ with this property of size $\geq(1-o(1))N$?

## Source ambiguities

The page does not specify the density convention, whether repetitions among the $a_i,b_j$ are allowed, or the ranges of $r,s$. The natural nondegenerate reading takes $r,s\geq1$ and allows elements of $A$ to be selected according to whatever convention the original source intended. Both repetition conventions should be analyzed until one is justified. The infinite target is “for every $\epsilon>0$, density $>1-\epsilon$,” not automatically one density-one set.

## Initial goal

Find asymptotically full finite constructions and, for every $\epsilon>0$, an infinite construction of density $>1-\epsilon$ with product-length rigidity.
