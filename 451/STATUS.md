# Status: Erdős #451

## Goal

Estimate the least $n_k$ via explicit constructions and unavoidable prime-divisor obstructions.

## Context

$n_k$ is the least integer above $2k$ for which every integer in $[n_k-k,n_k-1]$ has no prime divisor in the open interval $(k,2k)$.

## Constraints

No web search. Prime factors in the open interval $(k,2k)$ are forbidden across every factor $n_k-i$.

## Deliverables

Equivalent congruence/covering formulations, exact values for a useful initial range, and asymptotic upper/lower routes.

## Done when

A nontrivial asymptotic bound is proved or an explicit family improves the baseline construction.

## State

Unstarted. Main claim open locally.
